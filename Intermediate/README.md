# Intermediate Python Mini Projects

This folder contains simple Python projects created to practice core programming concepts like file handling, loops, conditions, and functions.

---

## 1. Student Marks Management System
A simple Python program that allows users to:
- Add student records (name and marks)
- View all students
- Search for a student
- Calculate the average marks

**Concepts Used:** File handling, loops, conditions, functions.

---

## 2. To-Do List App
A command-line Python app to manage daily tasks.

**Features:**
- Add a new task  
- View all tasks  
- Mark a task as completed  
- Delete a task  

**Concepts Used:** File handling, string operations, conditional logic.

---

## 3. Expense Tracker
A Python program to record and manage daily expenses.

**Features:**
- Add new expense (category, amount)
- View all expenses
- View total amount spent
- Search by category


**Concepts Used:** File handling, loops, math operations.

---

## Purpose
These projects help strengthen logical thinking and Python fundamentals, preparing for Django and React full-stack development.
